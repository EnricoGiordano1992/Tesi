Overview

This documentation provides a Doxygen generated output of the WAB-GW example applications and drivers.  Pages have been provided detailing the API for:

  • AtCmdLib – Implementation details of the GainSpan WAB Serial to Wi-Fi interface
  • Example Applications – Top level routines for controlling applications
  • Test Applications – Top level routines for doing various tests
  • Drivers – Various other drivers specific to the platform

Click on Modules for a complete list of the APIs available.  Clicking on each of the modules will provide a list of function to call in your code.  Additionally, the Data Structures tab shows a list of all structures used across all the code.  Finally, the Files section provides the same functions broken down by their .c and .h file location.
